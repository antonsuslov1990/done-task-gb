Решение тестовой задачи на курсе "Разработчик"

Я написал программу, которая из имеющегося массива строк формирует массив из строк, длина которых меньше либо равна 3 символа. Первоначальный массив можно ввести с клавиатуры, либо задать на старте выполнения алгоритма.

Создал массив с типом данных "строка", а дальше с помощью циклов(можно сделать с помощью методов и фкункции)сделал, так что каждый элемент в массиве переберается по отднльности и с помощью прооверки за ранее заданной, создается новый массив со элементами у которых всего 3 элемента, не больше, а другие отсеиваются.

И после полученный массив, выводим в консоли для пользвоателя.
